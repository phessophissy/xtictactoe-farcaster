# Animation Guide

## Core Animations

### Glow Pulse
Creates a pulsing green glow effect
- Class: animate-glow-pulse
- Duration: 2s
- Use: CTA buttons, winner indicators

### Leaf Sway
Subtle swaying motion like leaves
- Class: animate-leaf-sway
- Duration: 4s
- Use: Decorative elements

### Forest Shimmer
Shimmering effect with green gradient
- Class: animate-forest-shimmer
- Duration: 2s
- Use: Loading states, highlights

### Leaf Fall
Falling leaf animation
- Class: animate-leaf-fall
- Duration: 10s
- Use: Background decoration

### Bounce In Green
Entry animation with bounce
- Class: animate-bounce-in-green
- Duration: 0.5s
- Use: Modal/toast appearance

## Loading Animations

### Spinner
Rotating circle
- Class: loader-leaf-spinner

### Dots
Bouncing dots
- Class: loader-leaf-dots

### Pulse
Expanding circle
- Class: loader-leaf-pulse

### Bar
Progress bar
- Class: loader-leaf-bar

### Skeleton
Placeholder shimmer
- Class: loader-leaf-skeleton

## Usage Tips

1. Use animations sparingly
2. Respect prefers-reduced-motion
3. Keep durations under 3s for interactions
4. Use for feedback, not decoration
